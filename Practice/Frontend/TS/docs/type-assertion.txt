Type Assertion  "!"

 DOM
  - use an exclamation mark '!' at the end of an element declaration to surpass 'null'
  - Only use '!' when u're sure the value is not a null.
    const btn = document.getElementById("myBtn")!;
    btn.style.backgroundColor = "red";
